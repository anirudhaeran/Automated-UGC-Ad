# Automated-UGC-Ad
AI-Powered UGC Video Ad Generator An automated n8n workflow that generates realistic, selfie-style UGC (User Generated Content) video ads using Google Sheets as a control center. Integrates Veo 3.1, Sora 2, and NanoBanana (via Kie.AI) with GPT-4o vision analysis to turn product photos and text descriptions into high-converting video assets.
